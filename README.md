# Callers AI - One Brain Demo

Interactive demo showing how Callers AI enables proactive customer communication for airlines.

## [Live Demo](https://likhonroy013.github.io/callers-ai-demo/)

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/likhonroy013/callers-ai-demo)

## Features

**Manager Console (Left)**
- Real-time flight status dashboard
- One-click AI assistance trigger
- Live stats: active flights, delays, AI handling, resolved

**Passenger View (Right)**
- Simulated phone conversation
- Shows actual AI agent interaction
- Natural rebooking flow with confirmations

## Quick Start

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
npx vercel
```

## Tech Stack

- React 18
- Vite
- Lucide Icons
